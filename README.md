# Movie Ticket Reservation System

A Node.js backend project demonstrating client-server architecture, file handling, HTTP modules, Express framework, routing, middleware, and static file serving.

## Features
- View available movies
- Book tickets with seat selection
- Real-time seat availability
- File-based data storage
- RESTful API endpoints
- Static HTML pages


## API Endpoints
- GET `/api/movies` - Get all movies
- GET `/api/movies/:id` - Get movie by ID
- GET `/api/bookings` - Get all bookings
- POST `/api/bookings` - Create new booking
- GET `/api/bookings/:id` - Get booking by ID

## Access
- Home: http://localhost:3000
- Movies: http://localhost:3000/movies.html
- Booking: http://localhost:3000/booking.html
- API: http://localhost:3000/api
